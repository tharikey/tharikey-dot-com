ThariKey is a Thaana input method for macOS. Getting going takes three steps.

## 1. Install the app

Download ThariKey from the [download page](/download/) and move it to your Applications folder.

## 2. Enable the input method

Open **System Settings → Keyboard → Input Sources**, add **ThariKey**, and select it. macOS handles
the final selection — ThariKey registers and guides you there.

## 3. Pick a layout and type

Choose a keyboard layout in the app, then switch to ThariKey from the input menu and start writing
Thaana in any app. Unmapped keys pass through unchanged.

Stuck? The [download page](/download/#whats-new) has the current release notes, and you can open an
issue on [GitHub](https://github.com/tharikey).
